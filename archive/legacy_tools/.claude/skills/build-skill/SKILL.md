# Build Skill

## Purpose
This skill should execute the plan created by the Spec Skill.

## Instructions
1. Read the spec carefully.
2. Follow the build steps in order.
3. Make only the changes required by the scope.
4. Avoid unnecessary rewrites.
5. Keep code clean, simple, and consistent with the existing project.
6. Run relevant checks if available.
7. Summarize what was changed.
8. Mention anything it could not complete.

## Rules
- Do not add random features.
- Do not change unrelated files.
- Do not skip the spec.
- Do not mark the task complete unless the success criteria are met.

## Output Format
- Changes made
- Files changed
- Tests/checks run
- Any issues found
- Ready for review: yes/no
